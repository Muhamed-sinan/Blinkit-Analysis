# 🛒 Blinkit Sales & Operations Analysis Dashboard

## 📌 Project Overview

This project is an interactive Excel dashboard built using sales data inspired by Blinkit (India’s Last Minute App).

The objective of this analysis is to evaluate sales performance, outlet distribution, product trends, delivery efficiency, and customer behavior through meaningful KPIs and visual insights.

The dashboard provides a complete overview of business performance across outlet sizes, locations, and item categories.

## 📈 Analysis & Insights

### 🔹 Outlet Insights
- Tier 3 locations generated the highest revenue.
- High outlet size contributed significantly to total sales.

### 🔹 Product Performance
- Fruits & Vegetables and Snack Foods are top-performing categories.
- Seafood and Breakfast contribute the least revenue.

### 🔹 Fat Content Analysis
- Majority of products fall under Low Fat category.
- Regular products contribute a smaller share compared to Low Fat.

### 🔹 Outlet Type Performance
- Supermarket Type 1 dominates total sales.
- Grocery stores show moderate but stable performance.
